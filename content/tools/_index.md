# Werkzeuge

Die easydb bietet die folgenden Werkzeuge an:

- [CSV-Importer](csvimport): ermöglicht den Import von Daten im CSV-Format
- [JSON-Importer](jsonimport): ermöglicht den Import von Daten im JSON-Format
- [Rechte Im-/Export](rightsimexport): ermöglicht das Sichern und Einspielen von u.a. Gruppen, Pools, Tags und den dazugehörigen Berechtigungen
- [easydb 4 Migration](easydb4migration): ermöglicht die Übernahme von Daten aus easydb 4 in die aktuelle Version 5
