#  EAS-API: /batch

##  Example

```url
 http://eas.example.com/eas/batch?instance=example&batch={"commands":[{"command":"put","args":{"filename":"/tmp/foo.png"}}]}
```


##  Parameter


|key|value|
|---|---|
|`instance`          |Name of the Instance|
|`batch`             |Batch processing instructions (Documentation will follow)|
