#  EAS-API: /batch

##  Beispiel

~~~
http://eas.example.com/eas/batch?instance=example&batch={"commands":[{"command":"put","args":{"filename":"/tmp/foo.png"}}]}
~~~


##  Parameter


|---|---|
|`instance`          |Name der Instanz|
|`batch`             |Stapelverarbeitungsanweisungen (Dokumentation folgt)|
